# Train Benchmark for Typescript

A clone of the [Train Benchmark](https://github.com/ftsrg/trainbenchmark) in typescript.

### How to use
First clone the repo, then run:
```
npm install
```
To build the typescript:
```
npm build
```
To run the benchmark run:
```
node bin/index.js -f "this/is/the/location/of/the/benchmark/config"
```
